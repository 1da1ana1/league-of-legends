# League of Legends Champions

League of Legends Champions é um **site informativo** que apresenta campeões do jogo League of Legends, suas habilidades e características.  
Link para acesso online: https://league-of-legends-chi.vercel.app/index.html
- <img width="1200" height="758" alt="Captura de tela 2025-11-03 160309" src="https://github.com/user-attachments/assets/a4d3110d-d67a-4253-90fa-086b79fe714e" />

## Funcionalidades

- Listagem de campeões com imagens e descrições
- Visualização detalhada de habilidades e stats de cada campeão
- links com redirecionamento para o site oficial, com informações dos campeões
- foco no usuário, interface limpa
- respeitando a semântica de html

## Tecnologias e Skills Aplicadas
- **Linguagens:** JavaScript, HTML5, CSS3
